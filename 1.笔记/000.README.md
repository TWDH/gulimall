# Mall System

## Overview

- Backend: Springboot + Mybatis + Vue 
- Frontend: Springboot + Thymeleaf
- Tech Stack: Nginx + SpringCloud(Nacos + Feign) + Mysql + ElasticSearch + RabbitMq + Redis
